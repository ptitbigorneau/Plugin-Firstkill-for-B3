# firstkill plugin
# Plugin for B3 (www.bigbrotherbot.com)
# www.ptitbigorneau.fr

firstkill plugin (v1.5) for B3

Installation:

1. Place the firstkill.py in your ../b3/extplugins and the 
firstkill.ini in your ../b3/extplugins/conf folders.

2. Open your B3.xml file (default in b3/conf) and add the next line in the
<plugins> section of the file:

<plugin name="firstkill" config="@b3/extplugins/conf/firstkill.ini"/>
